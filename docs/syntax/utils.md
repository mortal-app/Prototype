## `path(name:string)`
Example

```php 
path('foo');  // /foo/bar
```

Source

 ```php 
/**
 * path
 *
 * @param  mixed $name
 * @return string $path
 */
function path(String $name) {
    global $routes;
    if (isset($routes[$name]['path'])) {
        return $routes[$name]['path'];
    } else {
        throwError('500', 'Sorry the path "' . $name . '" does not exist.');
    }
}
```

## `redirect(name:string, pathName:bool, params:array|null)`
Example

```php 
redirect('/foo/bar', false, ['id'=>1]); // redirect > /foo/bar?id=1
redirect('foo', true, ['id'=>1]); // redirect > /foo/bar?id=1
```

Source

 ```php 
/**
 * redirect
 *
 * @param  string $name
 * @param  array|null $params
 * @param  bool|null $pathName
 * @return redirect $path
 */
function redirect(string $name, bool|null $pathName = true, array|null $params = []) {
    if ($pathName) {
        $path = path($name);
    } else {
        $path = $name;
    }

    if ($params != []) {
        $query = http_build_query($params);
        header("Location: {$path}?{$query}");
    } else {
        header("Location: {$path}");
    }
    exit;
}
```

## `asset(name:string)`
Example

```php 
asset('img/foo.png');  // /static/img/foo.png
```

Source

 ```php 
/**
 * asset
 *
 * @param  string $name
 * @return string $path
 */
function asset(string $name) {
    return '/../../static/' . $name;
}
```
